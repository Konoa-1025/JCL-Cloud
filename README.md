# 🌸 JCL Cloud - 日本語プログラミング体験プラットフォーム

**ブラウザで気軽に始める、日本語プログラミングの世界へようこそ！**

## 🎯 何これ？

「プログラミングって英語ばっかりで難しそう...」そんな人でも大丈夫！  
**JCL（Japanese Coding Language）** を使えば、`主関数`や`表示`みたいな日本語でプログラムが書けるんです。

このサイトでは、ブラウザ上でJCLコードを書いて、すぐに実行結果を見ることができます。  
環境構築とか面倒な設定は一切不要！今すぐプログラミングを体験できます。

## ✨ こんなことができます

### 🚀 **ブラウザで即実行**
- 面倒なソフトのインストールは不要
- コードを書いたらワンクリックで実行
- エラーも日本語で分かりやすく表示

### 🤖 **AI支援機能**
- 「計算プログラムを作って」って書くだけでコード自動生成
- 書いたコードを分かりやすく説明してくれる
- より良いコードの書き方をアドバイス
- ChatGPTとの連携で詳しい質問も可能

### 🎨 **使いやすいエディタ**
- Monaco Editor搭載でVS Codeみたいな快適な操作感
- JCL⇔C言語の変換結果をリアルタイム表示
- エディタサイズを自由に調整可能
- ダークモード・ライトモード切り替え

## 📝 JCLってどんな感じ？

### Hello Worldはこんな風に書きます
```jcl
主関数() {
    表示("Hello, World!改行");
    戻る 0;
}
```

### 計算もできます
```jcl
主関数() {
    整数型 a = 10;
    整数型 b = 20;
    整数型 合計 = a + b;
    
    表示("答え: 整数改行", 合計);
    戻る 0;
}
```

### 繰り返し処理も日本語で
```jcl
主関数() {
    整数型 i;
    
    繰り返し(i = 1; i <= 5; i++) {
        表示("カウント: 整数改行", i);
    }
    
    戻る 0;
}
```

### 💡 **使い方のコツ**
- **型指定**: `整数型 年齢 = 20;` のように型を明示
- **フォーマット指定**: `表示("年齢: 整数歳改行", 年齢);` で値を表示
- **条件分岐**: `もし(年齢 >= 20) { 表示("大人です改行"); }`
- **繰り返し**: `繰り返し(i = 1; i <= 10; i++) { ... }`

## 🔧 技術的な話

実際にはこんな感じで動いています：

1. **フロントエンド**: HTML/CSS/JavaScript + Monaco Editor
2. **バックエンド**: Python FastAPI + Docker
3. **実行環境**: セキュアなサンドボックス内でのコンパイル・実行
4. **AI連携**: OpenAI APIによるコード生成・説明・最適化

JCLコードは内部的にC言語に変換してからコンパイル・実行されます。
